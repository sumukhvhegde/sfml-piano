# SFML Piano

A simple piano application built with **C++ and SFML-3**.
Press keys on your keyboard to play piano notes.

- **White keys:** Tab, Q, W, E, R, T, Y, U, I, O, P, [, ], \
- **Black keys:** 1, 3, 4, 6, 7, 8, 0, -, Backspace
